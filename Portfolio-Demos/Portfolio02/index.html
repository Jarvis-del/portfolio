02 / Banking-Ledger

Banking Transaction System

Backend Developer · Double-entry ledger API
  • Built a production-grade double-entry ledger API where every transfer atomically creates a DEBIT on the sender and a CREDIT on the receiver within a single MongoDB ACID transaction.
  • Designed balances to be derived at query time by aggregating ledger entries, making every transaction permanently auditable.
  • Enforced idempotency on all transfer endpoints via a unique idempotency key, so clients can safely retry failed transfers without duplicate charges.
  • Secured all routes with cookie-based auth and a token blacklist in MongoDB to immediately invalidate sessions on logout.
  • Integrated email notifications on registration and after every successful transaction, with full validation and descriptive error responses.
